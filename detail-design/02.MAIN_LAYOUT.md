# LOADSTAR Explorer - 메인 화면 레이아웃

## 전체 구조 (Eclipse-style)

```
┌──────────────────────────────────────────────────────────────────┐
│  Menu Bar                                                        │
│  [File] [Edit] [View] [Navigate] [Tools] [Help]                  │
├──────────────────────────────────────────────────────────────────┤
│  Toolbar                                                         │
│  [대시보드][TODO][History][모니터링][GIT][로그][CLI] | 검색 | 알림   │
├──────────────┬───────────────────────────────────────────────────┤
│              │  Editor Tabs (다중 탭, 닫기 가능)                    │
│  Element     │  ┌──────┐┌──────────┐┌────────┐                   │
│  Tree        │  │M://root││W://auth/login││B://auth/login│        │
│  (Explorer)  │  └──────┘└──────────┘└────────┘                   │
│              ├───────────────────────────────────────────────────┤
│  M://root    │                                                   │
│  ├─ M://auth │  Tab Content                                      │
│  │  ├─ W://  │                                                   │
│  │  │  └B:// │  선택된 탭의 콘텐츠 표시                             │
│  │  └─ W://  │  - 요소 상세 (Map 흐름도 / WayPoint / BlackBox)     │
│  ├─ M://ui   │  - 기능 화면 (대시보드 / TODO / GIT 등)              │
│  │  ├─ W://  │                                                   │
│  │  ...      │                                                   │
│              │                                                   │
├──────────────┴───────────────────────────────────────────────────┤
│  Status Bar (checkpoint 상태 | 드리프트 경고 | 프로젝트 경로)        │
└──────────────────────────────────────────────────────────────────┘
```

## 1. Element Tree (좌측 패널)

프로젝트의 Map → WayPoint → BlackBox 계층 구조를 트리로 표시한다.

```
M://root
├─ M://root/auth
│  ├─ W://root/auth/login
│  │  └─ B://root/auth/login
│  └─ W://root/auth/session
│     └─ B://root/auth/session
├─ M://root/ui
│  ├─ W://root/ui/dashboard
│  └─ W://root/ui/sidebar
│     └─ B://root/ui/sidebar
└─ M://root/cli
   ├─ W://root/cli/cmd_create
   │  └─ B://root/cli/cmd_create
   └─ W://root/cli/cmd_checkpoint
      └─ B://root/cli/cmd_checkpoint
```

- **Map (M://)**: 폴더 아이콘 — 펼치기/접기
- **WayPoint (W://)**: 노드 아이콘 — STATUS 색상 표시 (IDL: 회색, PRG: 파랑, STB: 초록, ERR: 빨강)
- **BlackBox (B://)**: 박스 아이콘 — SYNCED_AT 만료 시 경고 표시
- 트리 노드 클릭 → Main Content에 해당 요소 상세 표시
- 트리는 `.loadstar/` 하위 MAP, WAYPOINT, BLACKBOX 파일을 파싱하여 CONNECTIONS 기반으로 구성

## 2. Menu Bar

애플리케이션 최상단 메뉴. 전역 기능에 접근한다.

| 메뉴 | 항목 |
| :--- | :--- |
| File | 프로젝트 열기, 프로젝트 정보, 종료 |
| Edit | 실행 취소, 다시 실행 |
| View | Element Tree 표시/숨기기, Status Bar 표시/숨기기 |
| Navigate | 주소로 이동 (Ctrl+G), 뒤로/앞으로 |
| Tools | CLI 실행 (F09), Checkpoint 실행, Monitor Snapshot |
| Help | 단축키 목록, LOADSTAR SPEC 참조, 버전 정보 |

## 3. Toolbar

Menu Bar 아래 아이콘 버튼 행. 자주 쓰는 기능에 빠르게 접근한다.

```
[대시보드] [TODO] [History] [모니터링] [GIT] [로그] [CLI]  |  [검색] [알림 뱃지]
```

| 버튼 | 연결 기능 | 동작 |
| :--- | :--- | :--- |
| 대시보드 | F01 | 대시보드 탭 열기 |
| TODO | F05 | TODO 목록 탭 열기 |
| History | F06 | TODO History 탭 열기 |
| 모니터링 | F07 | 모니터링 탭 열기 |
| GIT | F08 | GIT 이력 탭 열기 |
| 로그 | F10 | 로그 검색 탭 열기 |
| CLI | F09 | CLI 실행 탭 열기 |
| 검색 | F11 | Command Palette 팝업 (Ctrl+K) |
| 알림 | — | checkpoint 필요, SYNCED_AT 만료, OPEN_QUESTIONS 미결 수 표시 |

- 툴바 버튼 클릭 → 해당 기능 탭이 Editor 영역에 열림 (이미 열려 있으면 포커스 전환)

## 4. Editor Tabs (우측 탭 영역)

이클립스 에디터 영역과 동일한 다중 탭 구조.

### 탭 동작 규칙

- **트리에서 요소 더블클릭** → 해당 요소 탭이 열림 (Map / WayPoint / BlackBox)
- **툴바 버튼 클릭** → 해당 기능 탭이 열림 (대시보드 / TODO / GIT 등)
- **탭 닫기 (x)** → 개별 탭 닫기
- **동일 요소 재클릭** → 기존 탭으로 포커스 이동 (중복 열림 방지)
- **탭 내 링크 클릭** (예: WayPoint의 BLACKBOX 주소) → 새 탭으로 열림

### 탭 유형

| 유형 | 탭 제목 예시 | 내용 |
| :--- | :--- | :--- |
| Map | `M://root/cli` | 하위 WayPoint 흐름도 (F02) |
| WayPoint | `W://root/cli/cmd_create` | 상세 — IDENTITY, CONNECTIONS, TECH_SPEC, ISSUE (F03) |
| BlackBox | `B://root/cli/cmd_create` | 상세 — CODE_MAP, TODO, ISSUE (F04) |
| 대시보드 | `대시보드` | 프로젝트 개요 (F01) |
| TODO | `TODO` | TODO 목록 (F05) |
| History | `TODO History` | TODO 이력 (F06) |
| 모니터링 | `모니터링` | 드리프트 현황 (F07) |
| GIT | `GIT 이력` | 커밋 이력 (F08) |
| 로그 | `로그 검색` | 로그 조회 (F10) |
| CLI | `CLI` | 명령 실행 (F09) |

## 5. Status Bar

화면 최하단. 현재 상태 정보를 표시한다.

```
[checkpoint: 2026-04-06 14:30] | [드리프트: 3건 미처리] | [프로젝트: C:\project\.loadstar]
```

- checkpoint_needed.flag 존재 시 경고 아이콘 표시
- 마지막 checkpoint 시각
- 드리프트 미처리 항목 수
- 현재 프로젝트 경로

## 6. 화면 전환 방식

- SPA — 페이지 리로드 없이 탭 단위로 콘텐츠 전환
- **트리 더블클릭** → 해당 요소 탭 열림 (싱글클릭은 트리 선택만)
- **툴바 버튼 클릭** → 해당 기능 탭 열림
- **탭 내 주소 링크 클릭** → 새 탭으로 열림
- 모든 `://` 주소는 클릭 가능한 링크로 동작
- 브레드크럼은 각 탭 콘텐츠 상단에 표시: `M://root > M://root/cli > W://root/cli/cmd_create`

## 7. 단축키

| 단축키 | 동작 |
| :--- | :--- |
| Ctrl+K | 검색 (Command Palette) |
| Ctrl+G | 주소로 이동 |
| Ctrl+W | 현재 탭 닫기 |
| Ctrl+Tab | 다음 탭으로 전환 |
| Ctrl+Shift+Tab | 이전 탭으로 전환 |
